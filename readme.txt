== Installation ==

= Automatic installation =

Automatic installation is the easiest option - WordPress will handle the file transfer, and you won't need to leave your web browser. To do an automatic install of AppConsent CMP by SFBX, log in to your WordPress dashboard, navigate to the Plugins menu, and click "Add New".
 
In the search field type "AppConsent CMP by SFBX" and click "Search Plugins". Once you've found the plugin,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by simply Clicking "Install Now" button.

= Manual installation =

Manual installation method requires downloading the AppConsent CMP by SFBX plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
